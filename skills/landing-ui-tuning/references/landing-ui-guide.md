# Landing UI Guide

## Source of truth
- Refer to `reference/architecture/landing-guidelines.md` for visual rules.
- Update `reference/architecture/system-overview.md` if routes change.

## Section order
- Hero
- Summary
- Core strengths
- Projects
- Experience
- Education
- Awards
- Contact

## CTA rules
- Keep one primary CTA per section.
- Right-top "view all" buttons stay consistent across sections.

## Styling notes
- Night-sky background only (no gradients).
- White content blocks with restrained shadows.
- One accent color for primary buttons.
