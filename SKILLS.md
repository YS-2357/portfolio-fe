# 스킬 개요

이 프로젝트는 반복 작업을 스킬로 분리해 관리한다.  
각 스킬은 `skills/<skill-name>/SKILL.md`에 정의하는 것을 원칙으로 한다.  
현재는 인덱스만 정의하며, 실제 스킬 폴더는 필요 시 추가한다.

# 스킬 목록

- `portfolio-layout-refine`
  - 위치: `skills/portfolio-layout-refine/SKILL.md`
  - 설명: 포트폴리오 레이아웃/타이포/그리드를 정리하고 반응형을 개선한다.
- `content-placeholder-sync`
  - 위치: `skills/content-placeholder-sync/SKILL.md`
  - 설명: “수정중/제작중” 플레이스홀더를 실제 콘텐츠로 치환한다.
- `contact-section-update`
  - 위치: `skills/contact-section-update/SKILL.md`
  - 설명: 연락처(이메일/깃허브/링크드인/블로그/전화) 정보를 최신화한다.
- `rag-chat-setup`
  - 위치: `skills/rag-chat-setup/SKILL.md`
  - 설명: 이력서 RAG 챗 흐름(문서/청크/출처 정책)과 기본 API 연동 규칙을 정리한다.
