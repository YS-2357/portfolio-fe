### [5] 오늘 분석/실험 중 얻은 인사이트나 발견한 문제점은?


✍️ 답변:


```plain text
1. 생성 모델의 한계: 
Stable Diffusion은 텍스트 조건에 따라 제품 형태 자체를 새로 생성해버리는 경향이 있어, 실제 제품 이미지의 디테일(프린팅, 로고 등)을 보존하며 가상 착용을 구현하기에는 부적합. RemBG 등으로 외형을 고정시키고 배경만 생성하는 방식이 더 적절할 것.
2. GCP 리소스 제약: 
SD 3.5-medium 모델 사용 중 디스크 부족. 모델 캐시와 가상 환경 용량만 20GB 이상을 차지 GCP 환경에서 고사양 모델 실험에 제약.
3. 데이터 중복 문제: 
하나의 GCP 인스턴스에서 여러 명이 각자 작업하므로 모델 데이터 및 기타 파일이 중복 저장되어 디스크 용량 낭비가 발생. 따라서 하나의 공유 폴더에서 공동으로 작업해야 할 필요성.
```

