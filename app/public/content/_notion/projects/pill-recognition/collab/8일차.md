## 💡 Notes & Ideas


- **Distribution Focal Loss (DFL)** 개념 학습 및 코드 기반 실험 준비

```python
python
코드 복사
def distribution_focal_loss(preds, target):
    left_idx = target.floor().long()
    right_idx = left_idx + 1
    weight_left = right_idx.float() - target
    weight_right = target - left_idx.float()
    loss = - (weight_left * torch.log(preds.gather(1, left_idx.unsqueeze(1))) +
              weight_right * torch.log(preds.gather(1, right_idx.unsqueeze(1))))
    return loss.mean()


```
